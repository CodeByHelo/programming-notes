# Linguagem C – Apresentação e História

## O que é a Linguagem C

A linguagem C é uma linguagem de programação **estruturada**, **compilada** e de **baixo nível**, criada para permitir maior controle sobre o hardware, sem perder legibilidade e portabilidade.  
Ela é muito utilizada no desenvolvimento de **sistemas operacionais**, **drivers**, **sistemas embarcados** e aplicações que exigem alto desempenho.

---

## Origem da Linguagem C

A linguagem C foi criada em **1972** por **Dennis Ritchie**, nos laboratórios **Bell Labs**, nos Estados Unidos.

Ela surgiu como uma evolução da linguagem **B**, que por sua vez foi baseada na linguagem **BCPL**. O principal objetivo era desenvolver um sistema operacional eficiente e portátil.

---

## Linguagem C e o UNIX

A linguagem C ganhou destaque por ter sido usada na reescrita do sistema operacional **UNIX**.

Antes disso, sistemas operacionais eram escritos em **linguagem de máquina** ou **Assembly**, o que dificultava a portabilidade.  
Com o uso da linguagem C, o UNIX passou a poder ser adaptado para diferentes arquiteturas de hardware com poucas modificações.

---

## Características da Linguagem C

- Linguagem **estruturada**
- Código **portável**
- Execução rápida e eficiente
- Acesso direto à memória por meio de **ponteiros**
- Possui uma biblioteca padrão rica (`stdio.h`, `stdlib.h`, etc.)
- Muito próxima do hardware

---

## Importância da Linguagem C

A linguagem C influenciou diversas outras linguagens modernas, como:
- C++
- Java
- C#
- Python
- JavaScript

Além disso, muitos sistemas importantes ainda são escritos em C, como:
- Sistemas operacionais (Linux, partes do Windows)
- Compiladores
- Sistemas embarcados

---


A linguagem C é uma das linguagens mais importantes da história da computação.  
Ela combina eficiência, controle e portabilidade, sendo até hoje muito utilizada no ensino e no desenvolvimento de sistemas de alto desempenho.
